# AI Powered Foam Template for Knowledge Workers

**👋 Welcome!**

## my
The `my` folder implements the GTD (Getting Things Done) methodology for personal task and workflow management:

- [inbox](my/inbox.md) - Capture point for incoming information and tasks
- [todo](my/todo.md) - Organized tasks using the Eisenhower Matrix (A-D priorities)
- [daily](my/daily/@Journal%20📔.md) - Daily reflections and notes

This folder supports the complete GTD workflow from capture to completion, with dedicated spaces for contacts, meetings, places, and workflow management.

## second-brain
The `second-brain` folder implements the PARA method for knowledge management:

- [Projects](second-brain/projects/) - Active initiatives with clear timelines and goals
- [Areas](second-brain/areas/) - Ongoing responsibilities and key life domains
- [Resources](second-brain/resources/) - Reference materials and topic-based knowledge
- [Archives](second-brain/archive/) - Completed or inactive items for future reference

This structure follows the CODE workflow (Capture, Organize, Distill, Express) to effectively manage personal knowledge and information.
